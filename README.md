## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## How to test the APIs
go to devices-challenge and execute : npm run start:dev

use postman to test the APIs, for example :
Create Device :

1- Ajoutez des devices en indiquant leurs températures et leurs humidités.
POST : localhost:3000/devices

{
"deviceName" : "test",
"temperature": 19,
"humidity": 0.45
}

Get All Devices:

GET : localhost:3000/devices

2- Changez la valeur température des devices ayant un ID paire en 20 °C.
POST : localhost:3000/devices/updateTemperaturePair

3-Changez la valeur température des devices ayant une température de 20°C à 19°C.
POST : localhost:3000/devices/updateTemperatureToNineteen

4-Récuperez les noms des devices ayant une temperature supérieur ou égale à 19°C
GET : localhost:3000/devices/getTemperaturesBeyondNineteen

5-Calculez la moyenne des humidités des devices ayant une temperature supérieur à 19°C
GET : localhost:3000/devices/calculateHumidityBeyondNineteen

6-a- Récuperez la liste des devices ayant le confort hygrothermique.
GET : localhost:3000/devices/getListeComfort

6-b- Effacez les autres devices qui n'avaient pas le confort hygrothermique.
DELETE : localhost:3000/devices/deleteNonComfort
